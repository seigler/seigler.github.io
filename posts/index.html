  • Site redesign! - I rebuilt the site with a from-scratch Hugo theme. Styles are assembled from LESS with a Gulp task runner. Afterwards, static assets are revision-hashed, and markup is reformatted. Hugo Hugo is a super fast, easily installed static site generator. It’s a lot like Jekyll, GitHub’s Ruby-powered static site generator. But it brings with it a lot of the Golang philosophy, and unlike Jekyll, Hugo doesn’t let you twist its arm into doing a lot of extra stuff like image processing or stylesheet building.
